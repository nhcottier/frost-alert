# Frost Alert

Frost Alert is a focused iPhone app for growers, gardeners, orchards, vineyards, nurseries, and lifestyle blocks.

It shows clear frost risk guidance for saved growing locations, including expected overnight lows, likely frost periods, a three-night outlook, local notifications, widgets, and practical protection actions.
